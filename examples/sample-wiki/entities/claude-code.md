# Claude Code

Anthropic's CLI-native coding agent, positioned around direct terminal workflows,
multi-file editing, and long-form reasoning inside a repository.

## Key Capabilities

- Conversational coding workflow in the terminal
- Strong repository-wide reasoning and synthesis
- Interactive tool use for reading, editing, and running code
- Good fit for long-lived sessions where context accumulation matters

## Workflow Notes

Claude Code is often evaluated against [[github-copilot]] for agentic coding work.
Its perceived advantage is usually tied to stronger long-form reasoning and to how it
handles large [[context-window]] workloads.

## Relationships

- Part of the broader [[agentic-coding]] trend
- Compared directly in [[github-copilot-vs-claude-code]]
- Benefits from larger [[context-window]] budgets
